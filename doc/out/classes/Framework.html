Framework Class

Defined in: framework\framework-base.js:54
Module: framework-base

框架基类模块
提供一些基础的公共方法
所有模块都应继承自它
此基础模块将会接管所有子模块的展现工作
只需设置showType类型，就可以多种形态进行展现
目前支持弹窗与普通展现、无界面三种形式
此套架构的优点在于：解耦目前所有模块，功能上各模块互相提供API接口进行调用，各模块API由各自统一进行维护。
此外，由于采用CMD模式架构，所以理论上支持无限量模块数目进行加载。
